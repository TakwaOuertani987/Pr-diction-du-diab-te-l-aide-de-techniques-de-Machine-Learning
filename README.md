# Prediction-du-diabete-l-aide-de-techniques-de-Machine-Learning
Développement d’un modèle de prédiction du diabète à partir de données patient (cliniques et démographiques) utilisant KNN, Régression Logistique et SVM. Tech : Python, Pandas, NumPy, Scikit-learn, Seaborn, Matplotlib, Dataset Pima Indians Diabetes.
_Analyse exploratoire et prétraitement des données issues d’un dataset sur le diabète (nettoyage des valeurs manquantes, traitement des zéros, visualisation statistique et corrélations).

_Implémentation et comparaison de plusieurs modèles de classification supervisée : K-Nearest Neighbors (KNN), Régression Logistique, SVM.

_Optimisation des hyperparamètres (sélection du meilleur k pour KNN, validation croisée avec GridSearchCV).

_Évaluation des modèles via accuracy, confusion matrix, classification report, ROC curve et interprétation des résultats pour identifier les variables les plus corrélées à l’issue (Outcome).

_Visualisation et interprétation des résultats grâce à des heatmaps, boxplots, histogrammes et matrices de confusion.
